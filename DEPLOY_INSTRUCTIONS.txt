GITHUB DEPLOYMENT INSTRUCTIONS / INSTRUCTIONS DE DÉPLOIEMENT GITHUB

The website files have been moved to the project root for seamless deployment on GitHub Pages.
Les fichiers du site ont été déplacés à la racine du projet pour un déploiement fluide sur GitHub Pages.

Step 1: Create a repository on GitHub (e.g., "cabinetry-work").
Étape 1 : Créez un dépôt sur GitHub (ex: "cabinetry-work").

Step 2: Run these commands in your terminal:
Étape 2 : Exécutez ces commandes dans votre terminal :

git add .
git commit -m "Initial website deployment"
git branch -M main
git remote add origin https://github.com/[YOUR_USERNAME]/[YOUR_REPO_NAME].git
git push -u origin main

Step 3: In GitHub Settings -> Pages:
- Set source to "Deploy from a branch"
- Select "main" and folder "/ (root)"
- Click Save.
Étape 3 : Dans GitHub Settings -> Pages :
- Définissez la source sur "Déployer à partir d'une branche"
- Sélectionnez "main" et le dossier "/ (root)"
- Cliquez sur Enregistrer.

Your site will be online at: https://[YOUR_USERNAME].github.io/[YOUR_REPO_NAME]/
Votre site sera en ligne à : https://[YOUR_USERNAME].github.io/[YOUR_REPO_NAME]/
